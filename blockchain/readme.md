# `hyperledger` 文档
`Hyperledger`是一项开源协作，旨在推动跨行业区块链技术的发展。这是由`Linux`基金会主办的全球合作，包括金融，银行，物联网，供应链，制造和技术领域的领导者。

# `ethereum` 文档
以太坊是一个开源的有智能合约功能的公共区块链平台，通过其专用加密货币以太币（`Ether`）提供去中心化的虚拟机（“以太虚拟机” `Ethereum Virtual Machine`）来处理点对点合约。
